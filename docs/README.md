LLM（大規模言語モデル）の内部構造に関する書籍は増えていますが、本を読んだだけでは「分かった気になる」で終わりがちです。結局、手を動かさないと理解は難しいのですが、「どうやって手を動かせば良い」のでしょうか。

このプロジェクトでは、GPT-2 という具体的なモデルの内部を一つずつ実装・実験しながら、関連する知識を膨らませていくアプローチを取っています。一般論としての解説ではなく、実際に動くコードの出力を起点に「なぜそうなるのか」を考えることで、理解を深めることを目指しています。

例えば、残差接続の実験からベクトルの標準偏差の変化を観察し、LayerNorm が cos 類似度に与える影響を発見し、そこから文章レベルの埋め込み実験へ進み、最終的には RAG や専用埋め込みモデルとの関係まで一本の線でつながります。抽象的に「対照学習で埋め込みモデルを作る」と説明されるよりも、GPT-2 で実際にやってみて「うまくいかない」ことを体験してから「だから専用モデルが必要」と理解する方が、腑に落ちるはずです。

このドキュメント群は、そうした実践的な学習の記録です。
